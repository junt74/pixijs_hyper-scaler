# design — 作品設計・設定

**プログラミングとは独立した**、作品の世界観・キャラクター・ストーリー・ゲームシステムの設計資料です。

- **world/** … 世界観、用語集、時代設定、地理など
- **lore/** … ロア（神話・歴史・伝承・世界内の文献など）
- **characters/** … キャラクター設定、台本、関係図
- **story/** … シナリオ、ストーリー構成、イベント一覧
- **mechanics/** … ゲームシステムの仕様（プレイ仕様書）
- **levels/** … ステージ・マップ・ワールド構成の設計

Markdown（.md）で書くと差分管理しやすく、検索も楽です。
