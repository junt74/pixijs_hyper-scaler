# reference — 参照用・公式資料

ゲームエンジン・ライブラリ・ツール・**プラットフォーム**の**公式ドキュメント**や、よく参照する資料を置く場所です。

- **engine/** … 使用しているゲームエンジン（Unity, Unreal, Godot など）のマニュアル・クイックリファレンス
- **libraries/** … 利用ライブラリの API リファレンス・サンプル（ローカルコピーやリンク集）
- **platforms/** … 配信・ログイン・SNS 等のプラットフォーム公式ドキュメント（例: mixi2 developer platform）
- **tools/** … エディタ・アセット制作ツールなどのマニュアル

公式ドキュメントは URL だけをまとめた `links.md` を置いておく運用がおすすめです。
